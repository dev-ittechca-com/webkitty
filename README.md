# WebKitty

**Web Development shouldn't be hard.**

That's why  I built WebKitty, an all-in one tool for editing, testing and debugging HTML projects.


## Features

- Customizable theme
- Integrated preview
- Simulation of different hosts and browsers
- Fast file navigation
- Cross platform
